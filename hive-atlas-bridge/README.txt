Atlas-Hive Integration Bridge

These files would typically contain the necessary JARs from the Atlas Hive hook package:
- atlas-application-properties
- atlas-client-common
- atlas-client-v1
- atlas-client-v2
- atlas-common
- atlas-intg
- atlas-notification
- atlas-server-api
- hive-bridge

For a complete integration, download the official Atlas Hive hook distribution from:
https://atlas.apache.org/Downloads.html

For demonstration purposes, we're using direct API integration instead of the hook.
